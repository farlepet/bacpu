CMP Instruction
---------------

	Compare two values OR if a certain condition is met. If the outcome is TRUE,
the TRUE flag is set in FLG.

*** Instruction Info layout ***
				  7								  0
+-----------------+-------------------------------+
| Compare (1 bit) | Comparison/Condition (7 bits) |
+-----------------+-------------------------------+

Compare:
	0) CMP checks if a condition is TRUE
	1) CMP checks if a comparison is TRUE

Comparison/Condition:
	IF Compare == 0
		0) Carry == 1
        1) Memory Mode == 1
        4) TRUE == 1 (No reason to use this)
	IF Compare == 1:
		0) Zero -- Compares first value to 0
		1) Equal
		2) Greater Than
		3) Greater Than or Equal to
		4) Less Than
		5) Less Than or Equal to


*** Instruction Flow ***

CMP
    * Check that both registers are valid
        * False
            * Increase PC
            * Call exception
        * True
            * Check Compare bit
                * Compare == 0
                    * Check a condition (See above)
                    * Result -> TRUE
                * Compare == 1
                    * Compare two registers for certain condition (See above)
                    * Result -> TRUE
            * Increase PC


*** Instruction Format ***

	Instruction Format: OpCode with two registers
	First Register:     First Value / Value / Unused
	Second Register:    Second Value / Unused
